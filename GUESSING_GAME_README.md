# Number Guessing Game 🎮

A fun and interactive number guessing game built with Python. Perfect for beginners to learn Python basics!

## 📝 Description

This is my **first project** in my CSE learning journey. The game challenges you to guess a random number between 1-100. It's a simple yet engaging way to practice Python programming fundamentals.

---

## 🎮 How to Play

1. **Run the Program:**
```bash
   python guessing_game.py
```

2. **Gameplay:**
   - The computer picks a random number (1-100)
   - You have to guess the number
   - After each guess, you get feedback:
     - "Too low!" - Your guess is less than the number
     - "Too high!" - Your guess is more than the number
     - "Correct!" - You found the number!

3. **Win Condition:**
   - Successfully guess the number
   - The game shows how many attempts you took

4. **Bonus:**
   - After 5 attempts, you get a hint! 💡

---

## ✨ Features

- ✅ **Interactive Gameplay** - Fun and engaging experience
- 📊 **Attempt Counter** - Tracks how many guesses you made
- 💡 **Hints System** - Get a hint after 5 attempts
- ✔️ **Input Validation** - Only accepts numbers between 1-100
- 🎉 **Celebratory Message** - Special message when you win!
- 📈 **User-Friendly** - Clear feedback on every guess
